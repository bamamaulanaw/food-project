# Food Project

Food Project is a Flutter-based mobile application that demonstrates a simple food ordering interface.

## Description

This project focuses on UI implementation, navigation, and basic user interaction.

The application provides:

- Login screen (UI only)
- Food listing in grid view
- Food detail page
- Order action via WhatsApp integration

This project uses local data models and assets without backend integration.

## Requirements

- Flutter SDK >= 3.4.4 < 4.0.0
- Dart SDK (included with Flutter)

## Installation

1. Clone this repository
2. Install dependencies:

   ```bash
   flutter pub get
   ```

3. Run the application:

    ```bash
    flutter run
    ```
